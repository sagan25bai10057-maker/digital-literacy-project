📌 Project Overview
As a Student Digital Ambassador, this portfolio serves as a comprehensive resource to help peers navigate the digital world safely and professionally. It covers essential modules from digital tool awareness to cybercrime prevention.

📂 Repository Structure
This repository is organized into five main task folders as per the project requirements:

task-1-presentation/: Contains the Digital Literacy Awareness Infographic.

task-2-portfolio/: Screenshots of professional profiles (LinkedIn, GitHub, etc.).

task-3-platforms/: Proof of coding challenges and the Digital Literacy Awareness Quiz.

task-4-email-etiquette/: Professional email drafts and a social media etiquette checklist.

task-5-cybercrime/: A cybercrime case study and a detailed prevention guide.

report/: The final comprehensive project report in PDF/DOCX format.
